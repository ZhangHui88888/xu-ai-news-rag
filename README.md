## 🚀 快速开始

### 环境要求

**硬件：**
- CPU: 4核+（推荐8核）
- 内存: 8GB+（推荐16GB）
- 硬盘: 50GB SSD+

**软件：**
- JDK 17+
- Maven 3.8+
- MySQL 8.0+
- Node.js 16+
- Ollama Latest
- Docker（可选）

### 部署步骤

#### 方式一：Docker部署（推荐）

```bash
# 1. 克隆项目
git clone https://github.com/your-repo/xu-ai-news-rag.git
cd xu-ai-news-rag

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 设置密码

# 3. 一键启动
docker-compose up -d

# 4. 访问系统
# 前端: http://localhost
# 后端: http://localhost:8080
# n8n: http://localhost:5678
```

#### 方式二：传统部署

```bash
# 1. 初始化数据库
mysql -u root -p < init.sql

# 2. 启动后端
cd backend
mvn clean package
java -jar target/xu-news-rag.jar

# 3. 启动前端
cd frontend
npm install
npm run dev

# 4. 启动n8n
docker run -d --name n8n -p 5678:5678 n8nio/n8n

# 5. 启动Ollama
ollama serve
ollama pull qwen2.5:3b
ollama pull all-minilm-l6-v2
```
